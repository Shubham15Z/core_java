Q1. What is Java?
Ans: 
Java is a programming language and a platform. 
Java is high level, robust, partially object oriented and secure programming language.
It is widely used for building a variety of applications such as desktop, mobile and web-based application.

Q3. Editions of Java?
Ans:
There are four editions of java as
1) Java SE - Java Standard Edition
- It includes Java Programming APIs such as java.lang, java.io, java.util, java.mathc etc
- It includes core topics of Java as OOP, String, Class, Multithreading, I/O Stream, Collection ec.

2) Java EE - Java Enterprise Edition
- It is an enterprise platform that is mainly used to develop web and enterprise applications.
- It is build based on Java SE platform.
- It includes topics like Servlet, JSP, Web Services, JPA etc.

3)Java ME- Java Micro Edition
- It is a micro platform that is dedicated to mobile applications.

4) JAVAFX
- It is used to develop rich internet applications. It uses a lightweight user interface API.

Q5.What is JDK JRE JVM?
Ans: JDK
JDK is Java Development Kit. It is a software development envirnoment which is used to develop 
Java applications and applets.
It consists of JRE + development tools.

JRE:
JRE is a Java Runtime Environment. It is a set of software tools which are used for developing Java applications.
It is used to provide the runtime environment. It is the implementation of JVM.
It consists of set of libraries with JVM.

JVM:
JVM is an abstract machine. It doesnit exist physically like JDK and JRE.
It is a specification that provides a runtime environment in which Java bytecode can be executed.
It perfoms the maintasks as Loads code, Verifies code, Execures code and Provide runtime environment

Q6 What is bytecode?
Ans:
Java bytecode is the instruction set for the JVM.
As sson as java program is compiled, java bytecode is generated.
It helps java to achieve plafrom independence.

Q7 What is Source Code?
Ans:
Source code is the high-level code or assembly code that is generated by a human/programmer.
It is easy to read and modify.

Q8 What is Machine Code?
Ans:
Machine is a set of instructions that is directly machine-understandable and executable.
It is processed by the CPU. 
It is in binary format which is completely different from the byte code and source code.
It is obtained after compilation or interpretation
